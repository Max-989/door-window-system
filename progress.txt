## Codebase Patterns
- Django project root is `backend/`. Config files are in `backend/config/settings/`.
- Backend tests are run with `pytest` using `config.settings.test` settings.
- Linting uses `flake8` with configuration in `.flake8`.
- Frontend type checking uses `tsc --noEmit` via `npm run type-check`.
- Secret key environment variable is `DJANGO_SECRET_KEY` (changed from `SECRET_KEY`).
- Base config sets `DEBUG = False`, dev overrides to `True`.
- When overriding nested settings like REST_FRAMEWORK, use copy.deepcopy to preserve base configuration.
